# BugMagnet日本語版
BugMagnetは探索的テストのためのChrome/Firefox拡張機能です。このフォークでは、元の[BugMagnet](https://github.com/gojko/bugmagnet)を日本語化しました。

## インストール手順

## 使い方
右クリックをして「Bug Magnet」を選択し、入力したい文字列を選択するとその文字列が入力フォームに反映されます。
